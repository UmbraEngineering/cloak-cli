
# cloak-cli



### Install

```bash
$ npm instal -g cloak-cli
```

##### Install bash completion

```bash
$ cloak completion install
```



### Usage

##### create a new project

```bash
$ cd Projects
$ mkdir new-project
$ cd new-project
$ cloak init
```

##### create a new model in the project

```bash
$ cloak create model person
```

##### destroy a model

```bash
$ cloak destroy model person
```
